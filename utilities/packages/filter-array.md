**FilterArray** is a common function a common function that removes deplicate items from an array.

### Install

```text
yarn add @common-utilities/filter-array -D
```

### Usage

```text
filterArray(['test', 'test', 'foo', 'bar', 'biz'])
// ['test', 'foo', 'bar', 'biz'])
```
